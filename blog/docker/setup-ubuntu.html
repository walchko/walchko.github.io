Setup Docker on Ubuntu

Kevin J. Walchko, PhD

14 Sept 2019

The offical instructions are here for the Community Edition (CE). The basics are:

  1. sudo apt update
  2. Remove old version: sudo apt remove docker docker-engine docker.io containerd runc
  3. Get neede apts: sudo apt-get install apt-transport-https ca-certificates curl software-properties-common
    1. apt-transport-https: Allows the package manager to transfer files and data over https
    2. ca-certificates: Allows the system (and web browser) to check security certificates
    3. curl: This is a tool for transferring data
    4. software-properties-common: Adds scripts for managing software
  4. Get key: curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add –
  5. Add repo: sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"
  6. sudo apt update
  7. Install latest version: sudo apt install docker-ce docker-ce-cli containerd.io
    1. Apparently there were some name changes, docker is now docker-ce

To prevent issues with not being able to connect to the daemon, do:

sudo usermod -aG docker $(whoami)

Note, you may need to log out and back in again.
