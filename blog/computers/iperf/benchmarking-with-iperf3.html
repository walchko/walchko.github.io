iperf3

Kevin J. Walchko, PhD

2016-04-02

iperf3 allows you to benchmark network performance. Github

Server

Start the server:

[kevin@Dalek ~]$ iperf3 -s -p 8282

Client

Run the client to test the network:

[kevin@Tardis ~]$ iperf3 -c Dalek.local -p 8282
