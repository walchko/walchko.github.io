They have recently changed their python library, so now all it does is send a text string to their servers … kind of stupid. You have to write your own code to send audio. Their staff doesn’t seem very knowledgable on python, hence all of the need help banners on their issues github page.

Oh well, it is free and works pretty good, so I can’t complain too much.
